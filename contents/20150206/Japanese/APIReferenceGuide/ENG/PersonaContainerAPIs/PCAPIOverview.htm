Persona Container APIs are divided into two the following categories:

·         Cell Level APIs

·         Box Level APIs

  • Cell Level APIs

Overview

Cell Level APIs consist of functions, such as building authentication mechanism to authenticate an application, social graph, generate and manage Box, event handling messages, etc.

It is implemented in the form of a control object. The Cell level APIs are further divided into the following categories:

Overview

Placement and acquisition of files, create and manage folders (collection), enumeration of files and folders, setting and reference of the access can be controlled as if it were a regular file system.
