Configure Access Control

  • 概要

This API provides access control function

  • 必要な権限

write-acl

  • 制限事項

·         Not manuscript

·         When you make an ACL configuration, it is updated in the form that overwrites the existing ACL settings

·         Limitation of version V1.0

    • Function to cancel the setting ACL (deny)

    • Gets a list of privilege that can be set in the ACL
