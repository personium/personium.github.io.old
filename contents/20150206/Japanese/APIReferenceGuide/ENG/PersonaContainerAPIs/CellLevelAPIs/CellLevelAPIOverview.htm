概要

Cell Level APIs are applicable to all the Cells. These APIs helps in building the authentication mechanism to authenticate who should access the Cell and Box. Cell level APIs facilitates other features, such as social graph management and event handling massages.

These APIs consist of the following set of APIs:

·         Authentication

·         Access Control

·         Messaging

·         Event

·         Cell Control Objects
