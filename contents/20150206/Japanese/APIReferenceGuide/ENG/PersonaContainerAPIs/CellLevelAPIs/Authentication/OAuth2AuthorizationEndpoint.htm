OAuth2 Authorization Endpoint (__authz)

  • 概要

This API is an authorization endpoint of OAuth2 when PCS JS application is used in native applications

  • 必要な権限

No

  • 制限事項

Request specified query, 'dc_target' parameter of the request body is not supported

·         If you specify a 'dc_target', because the value of the 'Location' header of the response is greater than 4096 characters, it is an error in nginx.

·         In Internet Explorer, the maximum length of the URL is 2,048 characters and you cannot redirect correctly
