OAuth2 Token Endpoint (OAuth2 Token endpoint)

  • 概要

This API authenticates and authorizes tokens using the following methods:

Password authentication

Only token issuer cell should issue a valid cell local token, by account authentication.

If authentication fails, account will be locked for 1 second. During the account lock period if user send request for account authentication:

·         an authentication error response is returned

·         the account lock period is extended to another 1 second

Token authentication

Only token issuer cell should issue a valid cell local token, by using transformer cell token

Refresh token authentication

Issues a cell local token again by refreshing the old cell local token

CURL Command

Password authentication

curl 'https://fqdn/cellname/__auth' -X POST -v -k \
-d 'grant_type=password&username=user&password=password'

Token authentication

curl 'https://fndn/cellname/__auth' -k -v \
 -X POST -d 'grant_type=urn:ietf:params:oauth:grant-type:saml2-bearer&assertion=SAML token'

Refresh token authentication

curl 'https://fqdn/cellname/__auth'  -k -v \
 -X POST -d 'grant_type=refresh_token&refresh_token=refresh token'

+ password authentication schema authentication 

curl 'https://fqdn/cellname/__auth' -X POST -v -k \
-d 'grant_type=password&username=pcsuser&password=pass&client_id=\
https://fqdn/appcellname/&client_secret=Token that has been produced from Application cell'

+ Trans-cell issue token authentication token
