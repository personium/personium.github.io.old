Send Message

  • 概要

This API sends a message to a particular Cell to accept or reject the relation request.

  • 必要な権限

message

  • 制限事項

·         リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·         リクエストボディはjson形式のみ受け付ける

·         レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·         $formatクエリオプションにatom または xmlを指定した場合、エラーとはならないが、レスポンスボディのデータの保証はない

·         V1.1.2 previous

    • It is ignored if you specify in the request body ToRelation

    • The response body when an error occurs when you send the message "Reason" (detailed message) not supported
