        </D:principal>

        </D:grant>
    </D:ace>

Syntax

curl https://example.com/cell  -X ACL -v -k \
-H "Authorization: Bearer auth_token" \
-d '<?xml version="1.0" encoding="utf-8" ?>\
<D:acl xmlns:D="DAV:" xmlns:dc="urn:x-dc1:xmlns" xml:base="http://localhost:8080/dc1-core/acell1/__role/__/"> \
<D:ace><D:principal><D:href>doctor</D:href></D:principal>\
<D:grant><D:privilege><dc:box-read/></D:privilege>\
<D:privilege><dc:auth/></D:privilege></D:grant></D:ace></D:acl>'

CURLサンプル

curl -X ACL "https://fj.baas.jp.fujitsu.com/cellName" -d "<?xml version=\"1.0\" encoding=\"utf-8\" ?>\
<D:acl xmlns:D=\"DAV:\" xmlns:dc =\"urn: x-dc1: xmlns\" xml:base=\"https://fj.baas.jp.fujitsu.com/ \
cellName/__role/boxName/\">\
<D:ace><D:principal><D:href>RoleName</D:href></D:principal>\
<D:grant><D:privilege><dc:box-read/></D:privilege></D:grant></D:ace></D:acl>\
" -H "Authorization:Bearer token" -i -k –s

 

Where:

cellName = Cell Name where you will implement ACL

boxName = Box Name associated with Role Name

RoleName = Role that is assigned to the specified Cell

token = token value
