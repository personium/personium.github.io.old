  • CURL Command

Syntax

curl 'https://fqdn/cellname' -X PROPFIND \
-H 'Authorization:Bearer auth_token' \
-d '<?xml version="1.0" encoding="utf-8"?><D:propfind xmlns:D="DAV:"><D:allpop/></D:propfind>' \
-H 'Depth:1' -i -k

CURLサンプル

curl -X PROPFIND "https://fj.baas.jp.fujitsu.com/cellName/" \



-d "\<?xml version=\"1.0\" encoding=\"utf-8\"?><D:propfind xmlns:D=\"DAV:\"><D:allprop/></D:propfind>"\
  -H "Depth: 1" -H "Authorization:Bearer token" -i -k -s

 

Where:

cellName = Cell Name to get its property

token = token value
