Structure XML

The body in the XML is according to the following schema:

項目名

Namespace

Node Type

概要

備考

propertyupdate

D:

Elements

Represents the root of propertyupdate, remove the set and is the child

 

set

D:

Elements

Represents a property setting, prop multiple is a child at least one

 

remove

D:

Elements

Represents a property setting deletion, prop multiple is a child at least one

 

prop

D:

Elements

Represents a property value, any element of the plurality is a child at least one

set time: node name of the child is the key to perform

remove time: the Delete key as the name of the child node

リクエストサンプル

<D:propertyupdate xmlns:D="DAV:" 
    xmlns:dc="urn:x-dc1:xmlns"
    xmlns:Z="http://www.w3.com/standards/z39.50/">
    <D:set>
        <D:prop>
            <Z:Author>Author1 update</Z:Author>
            <dc:hoge>fuga</dc:hoge>
        </D:prop>
    </D:set>
    <D:remove>
        <D:prop>
            <Z:Author/>
            <dc:hoge/>
        </D:prop>
    </D:remove>
</D:propertyupdate>

  • CURL Command

Syntax

curl https://fqdn/cell -X PROPPATCH -v -k \
-H "Authorization: Bearer auth-token" \
-d '<?xml version="1.0" encoding="utf-8" ?> \
<D:propertyupdate xmlns:D="DAV:" xmlns:dc="urn:x-dc1:xmlns" xmlns:Z="http://www.w3.com/standards/z39.50/">\
<D:set><D:prop><Z:Author>${author1}</Z:Author><dc:hoge>${hoge}</dc:hoge></D:prop>\
</D:set><D:remove><D:prop><Z:Author/><dc:hoge/></D:prop></D:remove></D:propertyupdate>'
