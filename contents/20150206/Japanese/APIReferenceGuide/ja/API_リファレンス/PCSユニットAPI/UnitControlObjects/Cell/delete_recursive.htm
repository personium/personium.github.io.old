Cell再帰的削除

  • 概要

  • 指定されたセル配下の関連データを全て削除する
  • 削除対象のデータは下記API

o        Box

o        Account

o        Role

o        Relation

o        ExtCell

o        ExtRole

o        ReceivedMessage

o        SentMessage

o        Collection

o        AssociationEnd

o        EntityType

o        ComplexType

o        Property

o        ComplexTypeProperty

o        $links

o        ユーザデータ

  • 必要な権限

 

  • 制限事項

なし

ヘッダ名

概要

有効値

必須

備考

X-Dc-Version

Data Cloud APIの実行バージョン

有効なバージョン

×

指定がない場合、最新のAPIバージョンが指定される

X-HTTP-Method-Override

メソッドオーバーライド機能

任意

×

POSTメソッドでリクエスト時にこの値を指定すると、指定した値がメソッドとして使用される

X-Override

ヘッダーオーバライド機能

${上書きするヘッダー名}:${値}

×

通常のHTTPヘッダの値を上書きします。複数のヘッダを上書きする場合はX-Overrideヘッダを複数指定する

X-Dc-RequestKey

イベントログに出力するRequestKeyフィールドの値

半角英数、-(半角ハイフン)と_(半角アンダーバー)
最大128文字

×

指定がない場合、PCS-${UNIX時間}を設定する
V1.1.7以降で対応
