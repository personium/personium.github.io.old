  • Access-Control-Allow-Methods

in the API of all, if you specify the OPTIONS request method to return the Access-Control-Allow-Methods header to the response header.

In the Access-Control-Allow-Methods header, return the request method that is permitted in the request URL of API.

The following is an example of the response of the Access-Control-Allow-Methods.

Access-Control-Allow-Methods: GET, OPTIONS

  • Access-Control-Allow-Headers

only if all the API, you specify the Access-Control-Request-Headers in the request header, and returns the Access-Control-Allow-Headers.

In the Access-Control-Allow-Headers header, return the value that is specified in the Access-Control-Request-Headers.

The following is an example of a response if you specify the "sample" in the Access-Control-Request-Headers.

Access-Control-Allow-Headers: sample
