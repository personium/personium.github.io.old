Role一覧取得API

概要

既存のRole情報の一覧を取得する

制限事項

なし

OData 制限

·           リクエストヘッダーのAcceptは無視される

·           リクエストヘッダーのContent-Typeは全てapplication/jsonとして扱う

·           リクエストボディはjson形式のみ受け付ける

·           レスポンスヘッダーのContent-Typeはapplication/jsonのみをサポートし、レスポンスボディはjson形式とする

·           $formatクエリオプションは無視される

必要な権限

auth-read

リクエスト

リクエストURL

/{Cell_name}/__ctl/Role

メソッド
