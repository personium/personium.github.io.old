イベント受付 API

概要

リクエストされたイベントを受付ける

Supported Versions

1.1.7

制限事項

なし

イベントAPI共通

·           ログの出力設定は未サポート    

o     ログファイル名は"default.log"固定とする

o     ローテートは以下のデフォルト設定に従い行う

§      ローテート数:12世代

§      ローテートのサイズ設定:50MB

§      ローテート後の圧縮方式:zip

ログの出力レベルは"info"固定（INFO, WARN, ERRORすべて出力）とする

·           ログ出力設定の参照は未サポート

×

概要

有効値

必須

備考

level    

ログ出力レベル

"INFO"/"WARN"/"ERROR"のいずれか

○  

小文字の"info"/"warn"/"error"が指定された場合は、大文字として扱う。
ただし、小文字での指定は非推奨（deprecated）。  

action  

イベントのアクション

文字列型
桁数：0～51200 byte

○  

 

object

イベントの対象オブジェクト

文字列型
桁数：0～51200 byte

○  

 

result

イベントの結果

文字列型
桁数：0～51200 byte

○
