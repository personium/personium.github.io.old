EntityType $links登録API

概要

EntityTypeの$links情報を登録する

必要な権限

write

制限事項

なし

リクエスト

リクエストURL

EntityType

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/EntityType(Name='{entityType_name}')/$links/_AssociationEnd

AssociationEnd

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/AssociationEnd(Name='{AssociationEnd_name}', _EntityType.Name='{EntityType_name}')/$links/_AssociationEnd

メソッド

POST

リクエストクエリ

特になし

リクエストヘッダ

ヘッダ名

概要

有効値

必須

備考

Authorization

OAuth2.0形式で、認証情報を指定する

Bearer {TokenValue}

×

※認証トークンは認証トークン取得APIで取得したトークン

Accept

レスポンスボディの形式を指定する

application / json

×

省略時は[application/json]として扱う

リクエストボディ

項目名

概要

有効値

必須

備考

uri

linkするAssociationEndのuri

存在するAssociationEnd

○

 

リクエストサンプル

{"uri": "https://fqdn/cellname/boxname/odatacolname/$metadata/AssociationEnd(Name='assocName',_EntityType.Name=null)"}

レスポンス

ステータスコード

コード

概要

備考

204

正常に登録が完了した場合

 

400

リクエストボディがJSON形式でなかった場合 
必須項目が指定されていない場合

 

401

認証トークンが無効の場合

 

403

アクセス権限が不足している場合

 

404

存在しないCellを指定した場合
存在しないBoxを指定した場合
存在しないODataCollectionを指定した場合

存在しないEntityを指定した場合
有効値でないリクエストボディの場合

 

405

許可していないリクエストメソッドを指定した場合

 

409

既にAssociationEndのリンク情報が登録されている場合

CURLサンプル

CURLコマンド(UNIX)

EntityType


curl'https://fqdn/cellname/boxname/odatacolname/$metadata/EntityType(Name='entitytypeName')/$links/_AssociationEnd'\ 
-d '{"uri": "https://fqdn/cellname/boxname/odatacolname/$metadata/AssociationEnd(Name='assocName_link',_EntityType.Name=null)"}'
-X POST -v -H 'Authorization:Bearer auth_token' -H 'Accept:application/json'

AssociationEnd

-X POST -v -i -k -s -H 'Authorization:Bearer auth_token' -H 'Accept:application/json'
