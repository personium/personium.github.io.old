サービスコレクションソース削除API

概要

サービスソース情報を削除する

必要な権限

write

制限事項

なし

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{path}/{src_name}

 

パス

概要

備考

Cell_name

セル名

  

Box_name

ボックス名

  

resource_path

リソースへのパス

有効値 桁数:1～128
使用可能文字種
半角英数字、半角ピリオド(.)、半角アンダーバー(_)、半角ハイフン(-)

src_name

サービスソースの名前を指定

有効値(制限) 桁数:1～128
使用可能文字種
半角英数字、半角ピリオド(.)、半角アンダーバー(_)、半角ハイフン(-)
