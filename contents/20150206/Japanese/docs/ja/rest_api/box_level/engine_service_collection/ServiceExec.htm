サービス実行

概要

登録されたサービスを実行する

必要な権限

exec

制限事項

·           共通制限

o     なし

·           WebDAV制限

o     未稿

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{path}/{src_name}

 

パス

概要

備考

Cell_name

セル名

 

Box_name

ボックス名

 

resource_path

リソースへのパス

有効値 桁数:1～128
使用可能文字種
半角英数字、半角ピリオド(.)、半角アンダーバー(_)、半角ハイフン(-)

src_name

サービスソースの名前を指定

有効値(制限) 桁数:1～128
使用可能文字種
半角英数字、半角ピリオド(.)、半角アンダーバー(_)、半角ハイフン(-)

メソッド

GET / POST / PUT / DELETE

クエリ名

概要

有効値

必須

備考

dc_cookie_peer

クッキー認証値

×

Authorizationヘッダの指定が無い場合のみ有効

クッキーの認証情報を利用する場合に指定する

 

リクエストヘッダ

共通リクエストヘッダー

ヘッダ名

概要

有効値

必須

備考

X-Dc-Version

APIの実行バージョン

有効なバージョン

×

指定がない場合、最新のAPIバージョンが指定される

X-HTTP-Method-Override

メソッドオーバーライド機能

任意

×

POSTメソッドでリクエスト時にこの値を指定すると、指定した値がメソッドとして使用される

X-Override

ヘッダーオーバライド機能

${上書きするヘッダー名}:${値}

×

通常のHTTPヘッダの値を上書きします。複数のヘッダを上書きする場合はX-Overrideヘッダを複数指定する

X-Dc-RequestKey

イベントログに出力するRequestKeyフィールドの値

半角英数、-(半角ハイフン)と_(半角アンダーバー)
最大128文字

×

指定がない場合、PCS-${UNIX時間}を設定する
V1.1.7以降で対応
