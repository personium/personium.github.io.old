全文検索(q) クエリ

概要

一覧取得時に全文検索キーワードを指定する場合は全文検索(q)クエリを使用する

コンプレックスタイプのデータを含めて、全ての値が検索対象

 

※クエリを指定する際はURLエンコードが必要
※__updated, _publishedの値を指定する場合は、UNIX時間（"/Date()"の括弧内の数字）で指定
※_metadata内の項目は検索対象外

リクエストクエリ

q={SearchKeyword}

 

項目

概要

有効値

備考

{SearchKeyword}

検索文字列を指定する

桁数：1～255 byte

 

CURLサンプル

·           「ぽち」というキーワードを含むデータを取得する
q=ぽち

·           「ぽち」かつ「たま」というキーワードを含むデータを取得する
q=ぽち%20たま

検索の仕様

·           半角空白

o     区切り文字として扱う

o     "(ダブルコート)で囲んでも、一単語として扱われない

o     例) 以下の指定はどちらも 「ぽち」かつ「たま」というキーワードを含むデータを検索する

q=ぽち%20たま

q="ぽち%20たま"

·           半角英数字

o     キーワード内での部分一致の検索不可

o     大文字・小文字の区別は行わない

·           全角文字

o     部分一致の検索可

o     大文字・小文字の区別は行わない
