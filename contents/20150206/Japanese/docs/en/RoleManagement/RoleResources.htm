Role Resources

The Role Resource shows the kind of Role registered in the Cell. The URL of the Role Resource is used in the Principal setting of ACL.

The URL structure of the Role Resource is as follows:

${CellURL}/__role/${BoxName}/${RoleName}

 

Where:

CellURL: URL of the Cell where the specified Role belongs

BoxName: Box name specified for the particular Role. If the Role is not associated to any box, it gets associated with the Main Box (__).

RoleName: Role name

Sample

Roles associated with the Box
https://fqdn/cell1/__role/box2/doctor2

 

Roles not associated with the Box
https://fqdn/cell1/__role/__/doctor2
