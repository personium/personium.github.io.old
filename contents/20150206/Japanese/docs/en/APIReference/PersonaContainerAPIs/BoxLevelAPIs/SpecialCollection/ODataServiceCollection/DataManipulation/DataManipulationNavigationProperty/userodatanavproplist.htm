Treatment of a number

Fractional value (Edm.Single type)

Handling of the case to get the UserOData in JSON format are as follows:

·           The value of the fractional part of 5.0, etc. is 0, It will return as an integer value

·           Numerical value of four decimal places, and returned in three-point after rounding

Numeric (Edm.Double type)

※ treatment of type Double of the personium.io follows the specification of the Java Double

·           Handling of the case to get the UserOData in JSON format are as follows:

o     The value of the fractional part of 5.0, etc. is 0, I will return as an integer value

·           Information about what is returned

o     If the number of input values ​​at the time of registration is accurate to double or more, performs data registration and rounded to the precision to be managed as floats internally, but the range that does not occur informative drop on output In and outputs the converted fixed-point number representation

o     When using the input fixed-point number output, the identity of the original number and the number of inputs is guaranteed
