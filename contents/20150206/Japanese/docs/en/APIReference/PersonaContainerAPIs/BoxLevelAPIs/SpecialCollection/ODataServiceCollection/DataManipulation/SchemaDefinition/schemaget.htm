Get Schema ($metadata)

概要

This API gets the schema information

必要な権限

read

制限事項

·           Normally Response body will be in XML format, where Content-Type of the Response Header will be in application/xml format.

·           Another format of the Response body is json, and the Context-Type of Response Header is application/json

·           When the $format is atomsvc or Accept Header is application/atomsvc+xml, will return the Atom Service Document of Schema.

·           When the $format is not atomsvc but Accept Header is application/atomsvc+xml, will return the EDMX of user data.

·           No Complex Type as well as Documentation tag will be returned as they are not supported

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{odataname}/$metadata

メソッド

GET
