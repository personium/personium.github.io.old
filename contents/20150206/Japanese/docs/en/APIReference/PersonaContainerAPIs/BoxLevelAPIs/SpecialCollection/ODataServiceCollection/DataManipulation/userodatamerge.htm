Partial Update Entity

概要

This API updates the Entity part of the user data, Http MERGE action enables a client to request updates to only specific properties of an entity

必要な権限

write

制限事項

Common Limitations

None

OData 制限

·           'Content-Type' header of the request is treated as application/json

·           Request body is accepted only in json format

·           'Content-Type' header of the response only supports application/json. The response body appears in json format

·           If you specify an atom as a $format query option, it is not an error, there can be no assurance or correctness of the data in the response body

ユーザデータ制限事項

·           Check the scope of the Edm.DateTime type property is not done properly

·           Array of Edm.DateTime type is not supported

·           If you specify the SYSUTCDATETIME () the properties of the Edm.DateTime type, there are cases where system time to be set is different

o     During the initialization time of request body and DefaultValue, (__published, __updated are the latter case timing)

·           It is possible to create up to 400 DynamicProperty・DeclaredProperty・ComplexTypeProperty for each EntityType
