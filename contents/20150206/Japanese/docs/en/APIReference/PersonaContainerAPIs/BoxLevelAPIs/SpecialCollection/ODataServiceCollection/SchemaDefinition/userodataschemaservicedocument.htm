Document Acquisition Service

概要

This API gets the service document for the definition of the user data schema

必要な権限

read

制限事項

None

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{odata_collection_path}/$metadata

メソッド

GET

リクエストクエリ

Common Request Query

項目名

概要

書式

必須

有効値

dc_cookie_peer

Authentication cookie value

Authentication cookie value that is returned from the server during authentication

×

Valid only if there is no specification of the Authorization header

to specify if you want to use the authentication information of cookies

Summary

Valid value

Required

Remarks

Valid version

×

X-HTTP-Method-Override

Any

×

X-Override

Header override function

×
