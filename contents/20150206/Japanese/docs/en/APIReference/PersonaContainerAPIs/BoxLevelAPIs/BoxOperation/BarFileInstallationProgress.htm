Progress of Bar File Installation

概要

This API gets the metadata of the Box. The metadata includes the following information:

·           State of the Box
For example, installation result, progress rate, error messages, etc.

·           The three kinds of present state of the Box are:

o     Box is available

o     Box during the installation process

o     Processing of Box installation terminated abnormally

·           Schema URL of the Box

·           Date and time of Box

If you get the progress of Box installation, note that:

·           In Box installation, we have adopted the asynchronous communication method.
Therefore, calls to complete processing from reception at the client will be as follows:

 

Call example of Box installation: (if you have 30 seconds to poll on the client)

  1. Box installation acceptance

1        MKCOL / {cell} / {box}

  1. Box installation status check

2        In return> "processing" - GET / {cell} / {box}.

3        30 seconds polling

  1. In return> "processing complete" - GET / {cell} / {box}.

The polling process to completion by looping the processing of 2. Above.

必要な権限

read

制限事項

·           Box expiration date status of the installation is identifiable, up to 72 hours later (including abnormal termination) Box installation is complete

·           If the Box installation fails, ensure to refer to the log output on the EventBus
