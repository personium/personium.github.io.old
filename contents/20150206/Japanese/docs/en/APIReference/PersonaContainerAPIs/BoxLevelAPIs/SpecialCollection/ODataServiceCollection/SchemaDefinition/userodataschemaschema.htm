User-Defined Data Schema ($metadata)

概要

This API retrieves the schema information for performing the schema definition of the user data

必要な権限

read

制限事項

None

リクエスト

リクエストURL

/{Cell_name}/{Box_name}/{odata_collection_path}/$metadata/$metadata

メソッド
