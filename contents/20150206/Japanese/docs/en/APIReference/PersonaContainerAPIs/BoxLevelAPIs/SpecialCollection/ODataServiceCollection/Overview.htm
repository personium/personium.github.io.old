AssociationEnd

AssociationEnd is OData object to define the end point of the relationship between the EntityType (Association).

2.     To the other end of the EntityType that defines the relationship, I register the AssociationEnd. At this time, I set at the same time multiplicity of the EntityType one also (Multiplicity) anymore.

3.     For AssociationEnd you have registered, to register the link in the $ links.
