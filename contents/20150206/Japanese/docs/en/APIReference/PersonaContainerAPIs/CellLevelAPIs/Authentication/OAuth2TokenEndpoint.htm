OAuth2 Token Endpoint

概要

This API authenticates and authorizes token using the following methods:

Password authentication

Cell local token could be retrieved by account authentication, which will be acceptable only for the author cell of that token.

If authentication fails, account will be locked for 1 second.

§      During the account lock period if user sends request for account authentication:

§      regardless of validating the user name and password, an authorization error will be returned.

§      Another one more second the account will be locked, from the time of request.

Token authentication

Only token issuer cell should issue a valid cell local token, by using transformer cell token

Refresh token authentication

Issues a cell local token again by refreshing the old cell local token

必要な権限

None

制限事項

Basic authentication

Not implemented

CURLコマンド(UNIX)

Syntax

Password authentication

Sample CURL command where type is password:


curl 'https://fqdn/cellname/__auth' -X POST -v -k \   -d 'grant_type=password&username=user&password=password'

Token authentication


curl 'https://fndn/cellname/__auth' -k -v \ 
-X POST -d 'grant_type=urn:ietf:params:oauth:grant-type:saml2-bearer&assertion=SAML token'

Refresh token authentication

Sample CURL command for refresh token authentication:


curl 'https://fqdn/cellname/__auth'  -k -v \ 
-X POST -d 'grant_type=refresh_token&refresh_token=refresh_token'

Password authentication + schema authentication


curl 'https://fqdn/cellname/__auth' -X POST -v -k \ 
-d 'grant_type=password&username=username&password=pass&client_id=https://fqdn/appcellname/&client_secret=app_cell_token'

Trans-cell issue token+ authentication token


curl'https://fqdn/cellname/__auth' -X POST -v -k \ 
-d 'grant_type=urn:ietf:params:oauth:grant-type:saml2-bearer&assertion=SAML_token&dc_target=https://fqdn/cellname/'
