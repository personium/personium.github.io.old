OAuth2 Authorization Endpoint (__authz)

概要

This API is the OAuth2 authorization endpoint, for utilizing the personium.io on JS application and native application.

必要な権限

None

制限事項

On request query, specifying the [dc_target] parameter in request body (trans cell token) is not supported

·           When [dc_target] is specified, and since the value of [Location] in response header is more than 4096 characters, it will display an error on nginx.

·           In Internet Explorer, the maximum length of the URL is 2,048 characters and you cannot redirect correctly
