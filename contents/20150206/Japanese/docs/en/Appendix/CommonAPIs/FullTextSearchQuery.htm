Full-Text Search Query

概要

This query performs linguistic searches against text data in full-text indexes. It operates on words and phrases based on rules of a particular language such as English or Japanese.

·           When you specify a query must be URL encoded

·           __updated, _ If you want to specify a value of published, specify (numbers in parentheses "/ Date ()" time) UNIX

·           items in the _metadata are excluded from the search

·           Gets the data containing the keyword "Fine"

q=Fine

·           Gets the data containing the keyword "Fine" and "Arts"

q=Fine%20Arts

Search Specifications

·           Byte space

o     Treated as a delimiter

o     Even if they are enclosed in double quotes, they are not treated as one word. For example, searching for data that contains the keywords "Fine" and "Arts"

q=Fine%20Arts

q="Fine%20Arts"

·           Alphanumeric

o     Does not search partial keyword matches

o     Does not perform search for case-sensitive keywords

·           Double-byte character

o     Search accepts a partial match
