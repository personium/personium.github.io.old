$select Query

概要

If you want to specify the properties to be returned, use the $select query to return all the properties acquired if it is omitted, however, about the properties of any of the following if you do not specify the $ select, you will be returned without fail

·           __id

·           __published

·           __updated

·           __metadata

※ If you specify the name of the property that does not exist in the $ select, to ignore a specified item 
If NULL, then not be able to get the property value value of Dynamic property specified by the ※ $ select is 
※ The property name "' I specify without enclosed in (single quote) "

リクエストクエリ

$select = {propertyName}

The default is $select = *

 

Path

概要

{PropertyName}

Return the property name
if you want to specify more than one must be separated by commas

CURLサンプル

1.     Returns the Name and Category

$select=Name,Category

2.       Return all properties

$select=*
