$inlinecount Query

概要

If you want to specify the number of results obtained during the enumeration, use the $inlinecount query if it is omitted, does not include the number of results you get in response

有効値

Value

概要

Example

備考

allpages

Acquisition, including the number of results

$inlinecount=allpages

 

none

I do not get the number of results, including

$inlinecount=none
