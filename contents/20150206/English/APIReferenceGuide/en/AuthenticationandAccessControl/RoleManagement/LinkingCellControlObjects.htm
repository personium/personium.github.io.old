Linking Cell Control Objects

To configure cell control objects with each other, you should link these cell control objects with External Role, External Cell, Relation, Role, and Account.

There are the following two methods to link cell control objects:

    1. Cell control objects are created in advance and you need to link them using $links. For example, Role and Account are created in advance. These two objects can be linked using $links API.

    2. Cell control objects are simultaneously created and linked using $links. For example, while creating a Role in combination of Box, the Box object automatically gets associated with the Role.
