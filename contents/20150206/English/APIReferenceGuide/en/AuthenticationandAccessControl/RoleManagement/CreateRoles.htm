Create Role(s)

To create a Role:

    1. Go to the specified Cell and click Role tab. The Role list appears. This list displays the Role Names created for the specified Cell.

    2. Click the Plus icon. The Create Role window appears.

    3. Enter Name and select box. Click Create Role. The successful message appears.

 

For more information on how the Role is crated at the server side, see Create Role API.
