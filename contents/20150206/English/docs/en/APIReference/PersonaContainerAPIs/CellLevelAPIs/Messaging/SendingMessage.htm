Send Message

Overview

This API sends a message to a particular Cell to accept or reject the relation request.

Required Privileges

message

Limitations

Common Limitations

None

OData Limitations

·           'Content-Type' header of the request is treated as application/json

·           Request body accepts only json format

·           'Content-Type' header of the response only supports application/json. The response body appears in json format

·           If you specify the XML or atom as a $format query option, it is not an error, but there can be no assurance of the data availability/correctness in the response body

None
