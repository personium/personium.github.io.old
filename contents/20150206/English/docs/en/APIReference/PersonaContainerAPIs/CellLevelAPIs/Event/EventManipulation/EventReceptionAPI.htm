Event Reception API

Overview

This API accepts the requested event

Required Privileges

event

Limitations

None

Common Event API

·           Log output of the internal event is not supported

·           Log output configuration is not supported

o     Set the log file name as "default.log "

o     Rotate will be according to following default settings

§      Set rotate size: 50MB

♦     Configure the log output label to "info" (output for all INFO, WARN, ERROR).

·           Log output configuration reference is not supported

Request

Request URL

URL

Summary

/{cellname}/__event

Event reception end point

Request Method

POST

Request Query

None

Request Header

Header name

Summary

Valid value

Required

Remarks

X-Dc-Version

Executable version of the Data Cloud API

Valid version

×

If not specified, the latest version of API is specified

X-HTTP-Method-Override

Function method override

Any

×

If this value is specified at the time of the request in the POST method, the specified value is used as a method

X-Override

Header override function

{Name of the header to override} $: $ {value}

×

Overrides the value of the HTTP header.

If you want to override multiple headers, specify multiple X-Override.

X-Dc-RequestKey

Value of RequestKey field to be output to the event log

Alphanumeric characters: up to 128, _ (underscore), and - (hyphen)

×

If not specified then set PCS-${UNIXtime}

JSON

Summary

Required
