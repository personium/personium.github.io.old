Get Box URL

Overview

This API retrieves the Box URL. As schema authenticated application will access the resource with the access token, the access will be redirected to the Box URL of the schema authenticated application.

For the application that does not support schema authentication, the access will still be redirected to the Box URL, considering the application will provide the schema URL parameter.

The Box URL will be fetched on the basis of Schema URL if the Token in the Authorization Header is schema-Authorized.

Required Privileges

Access control of this resource depends on the ACL of Box route. You must meet the following point.

·           When the "requireSchemaAuthz" context of Box root ACL is not none （public、confidential), meaning it is schema authenticated.

·           User has the "read "permission for the Box root. (user authentication is not required for the public Box root)

For requireSchemaAuthz attribute of ACL, see the access control model "schema request permission level".
