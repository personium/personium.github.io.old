Log File Information Acquisition API

Overview

This API gets the log file information in the event log.

It holds the number of generations of log files that have been rotated. The maximum number of generation is 12.

If you exceed the maximum number of generations to hold rotate when the log file, the log file of the oldest is deleted.

Supported Versions

V 1.3.14

Event Common API

·           Log output of the internal event is not supported

·           Log output configuration is not supported

o     Set the log file name as "default.log "

o     Rotate will be according to following default settings

§      Set rotate size: 50MB

o     Configure the log output label to "info" (output for all INFO, WARN, ERROR).

·           Log output configuration reference is not supported
