Password Change

Overview

This API changes the account password

To change the password, see Update Account. To update the password, you should have the permission of the cell-level auth ACL.

Required Privileges

Auth authority for cell-level ACL.

Limitations

None

Request

Request URL

{Cell_name}/__mypassword

Request Method

PUT

Request Query

Query is ignored

Request Header

Header name

Summary

Valid value

Required

Remarks

Authorization

In the form of OAuth2.0, specify the authentication information

Bearer{TokenValue}

○

Authentication token obtained in the acquisition API

X-Dc-Credential

Password

String

○

Number of characters: 6-32

Character type: alphanumeric characters, - (hyphen) and _ (underscore)

Request Body

None

Request Sample

None

Response Code

204

Response Header

None

Response Body

None

CURL Command

Syntax


curl -X PUT 'https://fqdn/CellName/__mypassword' -v -k \
-H 'X-Dc-Credential: change_password' \
-H 'Authorization: Bearer token'
