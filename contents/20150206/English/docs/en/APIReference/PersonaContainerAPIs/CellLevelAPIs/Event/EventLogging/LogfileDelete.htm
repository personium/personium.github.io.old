Delete Log File API

Overview

This API deletes the existing log files. It cannot delete the most recent log file.

It holds the number of generations of log files that have been rotated. The maximum number of generation is 12. If you exceed the maximum number of generations to hold rotate when the log file, the log file of the oldest is deleted.
