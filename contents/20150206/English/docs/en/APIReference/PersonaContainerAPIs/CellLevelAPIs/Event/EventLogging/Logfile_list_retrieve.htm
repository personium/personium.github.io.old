Retrieve Log File list API

Overview

This API retrieves the event logfile list from specified URL

Supported Versions

V 1.1.7

Limitations

Not manuscript

WebDAV Limitations

·           Maximum depth of the hierarchy of collection is 50

·           The maximum value of the number of child elements under each collection is 10,000

·           For count of hierarchy

1)hierarchy            2)hierarchy          … 10000)hierarchy
box/WebDavCollection1/WebDavCollection2/…/WebDavCollection10000/WebDavfile

 

The collection at first layer under Box, is counted as first hierarchy, and following will be 2nd and 3rd hierarchy respectively.

Files under a collection could not be counted  as hierarchy.

Common Event API

·           Log output of the internal event is not supported

·           Log output configuration is not supported

o     Set the log file name as "default.log "

o     Rotate will be according to following default settings

§      Set rotate size: 50MB

o     Configure the log output label to "info" (output for all INFO, WARN, ERROR).

o     At the time of rotate, file name should be default.log. {Timestamp}. {timestamp} is numbered instantly at the time when it is rotated.
