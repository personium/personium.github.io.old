Register EntityType_ $links

Overview

This API creates a link between EntityType and AssociationEnd or between two AssociationEnds

Required Privileges

write

Limitations

None

Request

Request URL

EntityType

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/EntityType(Name='{entityType_name}')\

/$links/_AssociationEnd

AssociationEnd

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/AssociationEnd(Name='{AssociationEnd_name}',/

_EntityType.Name='{EntityType_name}')/$links/_AssociationEnd
