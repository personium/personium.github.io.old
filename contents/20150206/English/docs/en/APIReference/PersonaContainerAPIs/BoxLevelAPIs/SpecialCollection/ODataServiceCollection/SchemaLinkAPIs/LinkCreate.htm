Create Schema Link

Overview

This API registers the link information of user data schema

Required Privileges

write

Limitations

None

Request

Request URL

AssociationEnd

/{Cell_name}/{Box_name}/{odata_colleciton_path}/$metadata/AssociationEnd(Name='{AssociationEnd_name}', _EntityType.Name='{EntityType_name}')/$links/_AssociationEnd

Request Method

POST

Request Query

Item Name

Summary

Format

Required

Valid Value

dc_cookie_peer

Authentication cookie value

Authentication cookie value that is returned from the server during authentication

×

Valid only if there is no specification of the Authorization header

to specify if you want to use the authentication information of cookies

Request Header

Header name

Summary

Valid value

Required

Remarks

Authorization

In the form of OAuth2.0, specify the authentication information

Bearer{TokenValue}

○

Authentication token obtained in the acquisition API

Accept

Specifies the format of the response body

application/json

×

Default is treated as [application/json]

Request Body

JSON

Item Name

Summary

Valid Value

Required

Remarks

uri

Specifies URI of the OData resources

Number of digits: 1 - 1024

According to the format of URI scheme：http/https/urn

○

 

In the case of related: "*, *", the number of data to be linked up to 10,000, an error if it exceeds 10,000

Request Sample

{"uri":"Https://Fqdn/Cell/__ctl/Box ('Boxname')"

Response

Response Code

Code

Summary

Remarks

204

If the registration is completed successfully

Confirmed operation

400

If the request body is not a JSON format

If a required item is not specified

 

401

Authentication token is invalid

 

403

If the access rights are insufficient

 

404

If you specify the Cell that does not exist

If you specify a Box that does not exist

If you specify a ODataCollection that does not exist

If you specify the Entity that does not exist

If the request body does not contain valid value

 

405

Specifies, the request method does not allow

 

409

If the link information of AssociationEnd is already registered
