Entity Bulk Operations

Overview

Against an OData Entity, you can perform bulk operations. In other words, you can pack several API requests and send them across to the Web API service in one HTTP request.

You can even receive a single HTTP response with the response of all requests.

Required Privileges

·           Get a list of acquisition
read

·           Registration, update, and delete
write
