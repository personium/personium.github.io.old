List AssociationEnd $links

Overview

Retrieve the list of OData resources, those are linked with Association End.

It is possible to specify the following OData resources:

·           EntityType

·           AssociationEnd

Required Privileges

Not manuscript

Limitations

·           'Content-Type' header of the request is treated as application/json

·           Request body accepts only json format

·           'Content-Type' header of the response only supports application/json. The response body appears in json format

·           If you specify the XML or atom as a $format query option, it is not an error, but there can be no assurance of the data availability/correctness in the response body

Request

Request URL

Linking with EntityType

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd(Name='{AssociationEnd_name}',\

_EntityType.Name='{entitytype_name}')/$links/_EntityType

Or,

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd(Name='{AssociationEnd_name}'')/$links/_EntityType

Or,

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd('{AssociationEnd_name}')/$links/_EntityType

Linking with AssociationEnd

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd(Name='{AssociationEnd_name}',\

_EntityType.Name='{entitytype_name}')/$links/_AssociationEnd

Or,

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd(Name='{AssociationEnd_name}')/$links/_AssociationEnd

Or,

/{Cell_name}/{Box_name}/{Collection_name}/AssociationEnd('{AssociationEnd_name}')/$links/_AssociationEnd

CURL Command

Syntax

To fetch a list of Entities that are linked with the respective association


curl -X GET "https://fqdn/CellName/BoxName/CollectionName/$metadata/AssociationEnd\
(Name='AssociationEndName',_EntityType.Name='EntityTypeName')/$links/_EntityType" -v -k \
-H 'Authorization:Bearer token'

 

To fetch a list of associations that are linked with the respective Entity


curl -X GET "https://fqdn/CellName/BoxName/odataName/$metadata/EntityType(‘EntityTypeName’)\
/$links/_AssociationEnd"\
 -H "application/json"\
 -H "Authorization:Bearer token” –I –k –s
