Unit Level API

Unit Level APIは、Cell群をホストするユニットに属するAPI(Cellの生成や作成したCell群の管理)です。
これらAPIは原則としてCellから発行されたアクセストークンではアクセスできません。
（ユニットレベルのトークンへの昇格操作を経て利用可能）
